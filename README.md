# Projeto criado a partir do curso de Java + POO pelo professor Nélio Alves

Este projeto foi criado durante o curso de java + POO para reciclar conhecimentos básicos de Java, padrões de projeto e Spring Boot.
A finalida é construir um CRUD de clientes, produtos, categtorias, ordens de serviço e pagamento.
Implementando relacionamentos OneToMany, ManyToMany e OneToOne.

## Tecnologias:

- Java 25
- Spring Boot 4.0.1
- Banco de dados H2
- Postman

## Experiências adquiridas:

- Criar um CRUD com API em Spring Boot
- Utilização de Repository e Injeção de Dependências
- Camadas Services
- Annotations Spring Boot
- Tratamento de Exceções
- Utilização de Clean Code com Single Responsability
- Imnplementação de métodos Http Post, Put, Get e Delete.
